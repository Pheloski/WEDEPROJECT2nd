Mama B Bakes – Part 2 README
Project Overview
Mama B Bakes is a responsive bakery website created to showcase baked products, provide company information, and allow customers to submit enquiries online.
This part of the project focused on:
•	CSS styling
•	responsive design
•	typography
•	layout design
•	mobile responsiveness
________________________________________
CSS Styling Implementation
External Stylesheet
An external stylesheet named style.css was created and linked to all HTML pages.
The stylesheet uses:
•	consistent naming conventions
•	reusable CSS variables
•	grouped selectors
•	cascading styles
________________________________________
Base Styling
The website uses:
•	a pastel pink, cream, and brown colour palette
•	global typography settings
•	spacing and layout consistency
A CSS reset was implemented using:
*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}
________________________________________
Typography
Typography choices:
•	Headings: Elegant script style using Playfair Display
•	Body text: Sans-serif font using Nunito
Typography styling includes:
•	font-family
•	font-size
•	font-weight
•	line-height
•	letter-spacing
________________________________________
Layout Structure
CSS Flexbox and CSS Grid were used throughout the website.
Examples:
•	Flexbox for navigation and forms
•	CSS Grid for the products section
The layout was designed to:
•	be easy to navigate
•	remain visually balanced
•	support responsive resizing
________________________________________
Visual Styling
Visual styling techniques used:
•	gradients
•	box shadows
•	rounded corners
•	hover effects
•	focus effects
•	colour variables
________________________________________
Responsive Design
Breakpoints
Media queries were used to adapt the website for smaller screens.
Breakpoint used:
•	700px and below for mobile devices
Responsive changes include:
•	hamburger navigation menu
•	single-column product layout
•	responsive forms
•	resized typography
•	adjusted spacing
________________________________________
Relative Units
Relative units used:
•	rem
•	%
•	fr
These units improve responsiveness across different screen sizes.
________________________________________
Responsive Images
Responsive image techniques used:
•	srcset
•	sizes
•	max-width: 100%
•	height: auto
These techniques help images scale correctly on different devices.
________________________________________
Testing and Developer Tools
Google Chrome Developer Tools were used to:
•	test responsiveness
•	inspect CSS
•	preview different devices
•	test mobile layouts
Devices tested:
•	Desktop
•	Tablet
•	Mobile
